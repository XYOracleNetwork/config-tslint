# XYO Shared TS-Lint Configuration (config-tslint)

## Branches

### Master

## Description

The shared configuration file for all XY and XYO TypeScript projects. It is based off the airbnb style guide

## Usage

In tslint.json:

```json
{
  "extends": "@xyo-network/config-tslint"
}
```
